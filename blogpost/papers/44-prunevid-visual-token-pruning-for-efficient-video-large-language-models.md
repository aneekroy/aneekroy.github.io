# PruneVid: Visual Token Pruning for Efficient Video Large Language Models

**Authors:** Various  
**Venue:** ACL 2025  

## Abstract

Identifies static video regions with minimal temporal variation. Reduces redundancy via temporal token merging, then spatial clustering, followed by question-guided attention.
