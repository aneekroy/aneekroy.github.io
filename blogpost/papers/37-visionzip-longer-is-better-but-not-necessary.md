# VisionZip: Longer is Better but Not Necessary

**Authors:** Yang et al.  
**Venue:** arXiv 2024  

## Abstract

Training-free visual token compression. Identifies key tokens using attention patterns. 90%+ token reduction possible for simple tasks.
